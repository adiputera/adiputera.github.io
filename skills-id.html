Yusuf Adiputera Keahlian & Kompetensi Teknis

Penjelasan mendalam mengenai teknologi dan kerangka kerja yang saya gunakan untuk membangun aplikasi enterprise yang kokoh, terukur, dan berkinerja tinggi. Fokus utama saya adalah ekosistem Java dan platform SAP Commerce (Hybris).

Java

Lebih dari 8 tahun pengalaman membangun backend kritis. Mahir dalam Java 8 hingga Java 21+, memanfaatkan fitur modern seperti Virtual Threads (Project Loom) untuk mencapai konkurensi throughput tinggi.

Lead Level

SAP Commerce (Hybris)

Sebelumnya dikenal sebagai Hybris. Pengalaman mendalam dalam arsitektur platform, pemodelan data, pengembangan OCC API, dan kustomisasi Backoffice. Implementasi tingkat senior untuk platform commerce enterprise.

Lead Level

Spring Boot & MVC

Penggunaan luas ekosistem Spring untuk arsitektur microservices maupun monolitik. Pemahaman mendalam tentang Dependency Injection, Security, dan Data (JPA/Hibernate) untuk skala enterprise.

Advanced

Integrasi REST & SOAP

Spesialisasi dalam integrasi sistem kompleks menggunakan RESTful API dan layanan web SOAP. Fokus pada integritas data, komunikasi aman, dan pertukaran pesan yang andal.

Advanced

Desain Sistem & Performa

Fokus pada optimasi performa dan skalabilitas arsitektur. Implementasi strategi caching yang canggih menggunakan Redis, Ehcache, dan CDN untuk memastikan responsivitas sistem.

Advanced

Apache Kafka

Menerapkan arsitektur berbasis event (event-driven) untuk sinkronisasi data real-time. Berpengalaman merancang producer, consumer, dan manajemen topik untuk ketersediaan tinggi.

Intermediate

Cloud & Serverless

Berpengalaman dengan Azure Function Apps. Menerapkan logika serverless dan pipeline CI/CD untuk meningkatkan kapabilitas platform enterprise.

Intermediate

SAP CDP & Emarsys

Kemahiran tingkat Intermediate dalam SAP Customer Data Platform dan SAP Engagement Cloud (formerly SAP Emarsys) untuk mendorong keterlibatan pelanggan omnichannel dan otomatisasi pemasaran yang dipersonalisasi.

Intermediate

Drools

Pengalaman dengan Drools untuk manajemen aturan bisnis. Menerapkan pemisahan logika bisnis melalui rule-based engine yang fleksibel.

Intermediate

SAP S/4HANA ABAP

Familiar dengan pengembangan ABAP untuk ekstensi logika bisnis inti dalam ekosistem SAP S/4HANA.

Beginner

Filosofi Pemrograman

  • Clean Code: Mematuhi prinsip SOLID untuk memastikan pemeliharaan jangka panjang yang mudah.
  • Performa Utama: Optimasi proaktif dan profiling untuk memastikan waktu respons di bawah satu detik.
  • Mentorship: Memimpin dengan memberi contoh dan menumbuhkan budaya keunggulan teknis dalam tim engineering.
